Лабораторная работа №1

Управляющие конструкции

Внимание!

Создайте в репозитории директорию javascript. В этом каталоге размещайте файлы выполненных заданий, ссылки на которые добавляйте в index.html.

Обязательно использйуте директиву 'use strict'; в начале скриптов. Необходимо предерживаться хорошего стиля написания кода.

При работе в редакторе Visual Studio Code рекомендуется использовать плагины ESLint (линтер для JavaScript) и Live Server.

Задания

  1. Создайте переменные age и gender, запросите у пользователя их значение. Используя конструкцию if ... else напишите скрипт, который проверяет значение переменных age, gender и:
    • если значение age попадает в диапазон 0–17 (вкл.) выводит сообщение "Вам работать ещё рано — учитесь"
    • если значение age попадает в диапазон 18–59 (вкл.) для мужчин или в диапазон 18–54 (вкл.) для женщин выводит сообщение "Вам ещё работать и работать"
    • если значение age попадает в диапазон 60–64 (вкл.) для мужчин или в диапазон 55–59 (вкл.) для женщин выводит сообщение "Скоро пенсия!"
    • если значение больше 65 для мужчин или больше 60 для женщин выводит сообщение "Вам пора на пенсию"
    • в остальных случаях выводитсясообщение "Да кто ты такой?"
  2. Создайте переменную num и запросите у пользователя её целочисленное значение. Используя конструкции if ... else и switch, напишите скрипт, который проверяет значение в переменной num и выводит фразу "На ветке сидит [значение переменной num] ворон" с правильными окончаниями. То есть:
    • На ветке сидит 1 ворона
    • На ветке сидит 2 вороны
    • На ветке сидит 5 ворон
  3. Напишите цикл, который предлагает prompt ввести число, большее 100. Если посетитель ввёл другое число – попросить ввести ещё раз, и так далее. Цикл должен спрашивать число пока либо посетитель не введёт число, большее 100, либо не нажмёт кнопку Cancel (ESC).
  4. Натуральное число, большее 1, называется простым, если оно ни на что не делится, кроме себя и 1. Другими словами, n>1 – простое, если при делении на любое число от 2 до n-1 есть остаток. Создайте код, который выводит в консоль все простые числа из интервала от 2 до n. Значение n задаёт пользователь.
  5. Отрисуйте в консоли браузера таблицу вида: 
    dog dog dog cat cat dog
dog dog cat cat dog dog
dog cat cat dog dog dog
...
    Последовательность содержит три собаки, два кота. В каждой строке выводится 6 животных. Число строк задаёт пользователь. Разделитель - символ табуляции.
